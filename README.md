# Deep-Learning

A comprehensive collection of Deep Learning, NLP, and AI projects built using Python, TensorFlow/Keras, and Machine Learning techniques.

## Project Structure

```
Deep-Learning
├── SentimentAnalysis
├── MachineTranslation
│   ├── app.ipynb
│   ├── requirements.txt
│   ├── .gradio
│   └── finetuned-nlp-en-hi
└── README.md
```

## Projects

### 1. SentimentAnalysis

A deep learning-based sentiment analysis project that classifies Swiggy restaurant reviews as positive, negative, or neutral.

**Location:** `SentimentAnalysis/`

**Contents:**
- `app.ipynb` - Complete sentiment analysis implementation and model training
- `swiggy.csv` - Dataset with Swiggy reviews
- `requirements.txt` - Project dependencies
- `README.md` - Project-specific documentation

### 2. MachineTranslation

A machine translation project with a Jupyter notebook, saved translation model files, and Gradio support.

**Location:** `MachineTranslation/`

**Contents:**
- `app.ipynb` - Translation notebook
- `requirements.txt` - Project dependencies
- `.gradio/` - Gradio app settings and metadata
- `finetuned-nlp-en-hi/` - Fine-tuned NLP model files

## Installation

1. Clone the repository:
```bash
git clone https://github.com/rushipatil1808/Deep-Learning.git
cd Deep-Learning
```

2. Use the project-specific requirements file:
```bash
cd MachineTranslation
pip install -r requirements.txt
```

3. Launch the notebook:
```bash
jupyter notebook app.ipynb
```
