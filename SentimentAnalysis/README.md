# SentimentAnalysis Project

A sentiment analysis project built with Deep Learning techniques to analyze and classify sentiment from Swiggy restaurant reviews.

## Project Files

- **app.ipynb** - Jupyter notebook containing the main sentiment analysis implementation and model training
- **swiggy.csv** - Dataset containing Swiggy reviews for sentiment analysis
- **requirements.txt** - Python dependencies required for the project

## Requirements

Install the required packages using:

```bash
pip install -r requirements.txt
```

## Usage

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Open the notebook:
```bash
jupyter notebook app.ipynb
```

3. Run all cells to train and test the sentiment analysis model

## Installation

From the repository root:

```bash
cd SentimentAnalysis
pip install -r requirements.txt
jupyter notebook app.ipynb
```

## Model Details

This project implements a deep learning-based sentiment classifier trained on the Swiggy dataset to classify reviews as positive, negative, or neutral.
